    Moments and Stochastic Trees from Event Reactions

    Project maintained by Tim Vaughan Hosted on GitHub Pages — Theme by mattgraham

    MASTER is a Java package aimed at providing a general means of creating simulators for models in theoretical population genetics. Specifically, it is aimed at modelling the stochastic dynamics of structured and unstructured populations, and simulating inheritance trees and networks under these same models.

    System Requirements

    MASTER is a Java application, and is thus fundamentally platform-agnostic. In order to use it, v1.6 or later of the Java Runtime Environment is required. This probably exists on your system already, but if not you may obtain it here.

    Installation

    Installation of MASTER is very straight-forward. Simply extract the following archive:

    • Master-1.0.zip
    Once complete, the new folder "Master-1.0/" will contain a number of files. On systems running a modern Windows installation, double-clicking on "Master-1.0.jar" will bring up a file selector that you can use to select an XML input file describing a simulation. On OS X and Linux computers, the shell script "master.sh" can be used to run Master from the command line, with the path to an XML input file given as a command line argument. (In this case, making a symbolic link to master.sh from a directory in your execution path like "/usr/local/bin/" will allow you to execute Master on the command line easily from other locations in your file system.)
